Scientific Visualization of Simulation Data in VR

  • Stereoscopic rendering of 3D material simulation data
  • Simulatneously running on multiple displays of different kind: HMD VR, immersive VR, and desktop

This project focuses on cross-platform or cross-device visualization of scientific simulation data. Current data of interest is from dynamic MonteCarlo simulation by Van der Ven Group. The data with a three-dimensional parameter space (Temperature / Chemical Potential / Time step) is loaded and presented in different devices such as flat desktop computer screen, large scale immersive environment (AlloSphere), and head-mounted display for personal vritual reality (HTC Vive). Due to large volume of the data, decompressing the original gzip files and extracting only the relevant slice of data within them only when needed as the user explores the parameter space.

Work towards collaborative interaction on distributed systems. Support for displaying the datasets in the Vive HMD to use it and its controllers as a single user control device while others view the data on separate desktop/projector. This involved porting our interactive 3D pick-ray for use with VR controllers, as well as designing interaction modes for the headset. A person wearing the HMD can now more naturally rotate and understand the dataset structure, and has interactive control of the visualization and its parameters through the wireless devices.
